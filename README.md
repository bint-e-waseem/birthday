# 💝 birthday.cpp — A Friendship, Compiled

A heartfelt C++ program written as a birthday gift for a best friend. This isn't just code — it's a love letter in the form of a console application, complete with a login screen, ASCII art, and password-protected memories.

## 📖 The Story Behind the Code

This project started as an inside joke that turned into something meaningful. Instead of a traditional birthday card, I built a program that only my best friend could unlock. Every function, every line of code, and every message maps to something real:

- **6 years** of friendship captured in the `Friendship` class
- **A hand-drawn ASCII heart** built character by character with nested loops
- **An "infinite" friendship loop** that's secretly capped (because programs have to end, even when feelings don't)
- **A password-protected memory box** that only opens for someone who already knows the password

The program is a small, self-contained C++ application that delivers a personalized message through the terminal — because sometimes the most meaningful gifts come in unexpected forms.

## ✨ Features

### 🔐 Bestie Login Screen
- Username and password authentication before revealing anything
- Only the intended recipient can access the heartfelt messages
- Creates a sense of intimacy and personal connection

### 💛 The Friendship Class
- Bundles a friend's name and years of friendship into an object
- Displays a personalized opening message
- Includes an "infinite" friendship loop with visual feedback

### ❤️ ASCII Heart Generator
- Pure C++ with no external libraries
- Nested loops creating a beautiful heart shape
- Represents love and care through code

### 📦 Password-Protected Memory Box
- Secondary password gate for extra intimacy
- Displays shared memories and inside jokes
- Acts as a private easter egg within the program

### ✨ Interactive Web Version
- The project includes a companion webpage (`index.html`) that explains the story
- Features a live terminal replay showing the program in action
- Interactive heart button that lets users "send loops of love"
- Fully responsive design with accessibility considerations

## 🚀 How to Run

### Prerequisites
- C++ compiler (g++, clang++, or MSVC)
- C++17 or later

### Compile and Run
```bash
# Compile
g++ -std=c++17 -O2 -o birthday birthday.cpp

# Run (Linux/macOS)
./birthday

# Run (Windows)
birthday.exe
```

### Login Credentials
- **Username**: `BFF`
- **Password**: `MY GIRL`

## 🌟 About The Project

**Wish** is a heartfelt interactive program created to celebrate **6 years of friendship**. It combines a nostalgic C++ terminal experience with a modern web interface, all wrapped in a personal message of love and gratitude.

This project was born from a simple idea: *"How do I tell my best friend they mean the world to me?"* — The answer was code, creativity, and a whole lot of heart.

### ✨ The Story Behind It

Six years ago, two nervous girls met and became inseparable. Through college, fights, 3 AM talks, countless selfies, and life's ups and downs, their bond only grew stronger. This program is a digital love letter capturing that journey.

---

## 🎯 Features

### 💻 C++ Terminal Version (`birthday.cpp`)
- **Password-Protected Login** - Only true BFFs know the way in
- **ASCII Heart Generator** - A beautiful heart made of asterisks
- **Interactive Memory Box** - Unlock treasured memories with a secret password
- **Infinite Friendship Loop** - A symbolic representation of unbreakable bonds
- **Cross-Platform** - Works on Windows, Linux, and macOS

### 🌐 Web Version (`bestie_gui.html`)
- **Visual Login Interface** - Click-based tile selection (no typing)
- **Animated Heart** - Pulsing with love ❤️
- **Interactive Memory Cards** - Six years unfolding one by one
- **Responsive Design** - Works on desktop and mobile
- **Real-time Feedback** - Instant responses to user actions

## 🚀 Getting Started

### Prerequisites
- **C++ Version**: Any C++17 compiler (g++, clang, or MSVC)
- **Web Version**: Any modern web browser

### Installation & Running

#### Option 1: C++ Terminal Experience
```bash
# Clone the repository
git clone https://github.com/yourusername/wish.git
cd wish

# Compile the program
g++ -std=c++17 src/birthday.cpp -o birthday

# Run it!
./birthday
```

#### Option 2: Web Browser Experience
```bash
# Simply open the HTML file in your browser
open web-version/bestie_gui.html
# OR double-click the file
```

## 🎨 Design Philosophy

### Emotional Design
- Every element serves the story of friendship
- Colors chosen for warmth (pinks, golds, plums)
- Typography feels handwritten and personal
- Animations bring the experience to life

### Technical Choices
- **Text-based emojis** (`[HEART]`) instead of actual emojis - ensures cross-platform compatibility
- **ASCII art** for the heart - works in every terminal
- **Tile-based input** in HTML - more engaging than typing
- **Progressive disclosure** - reveals content gradually for emotional impact
